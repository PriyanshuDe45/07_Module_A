Image Mosaic API

Express server that returns a pixelated mosaic version of any image in the resource/ folder.

Setup

npm install
mkdir resource
# drop your images into ./resource/
node server.js

Endpoint

GET http://localhost:3004/mosaic

Query Parameters

ParamDefaultDescription
imageoriginal.jpgFilename inside resource/
cell_size50Size of each mosaic cell in px

📬 Example URLs

http://localhost:3000/mosaic
http://localhost:3000/mosaic?cell_size=20
http://localhost:3000/mosaic?image=sample.png&cell_size=30
